# Lazy vim based config
Based on modular kickstarter config.

https://github.com/dam9000/kickstart-modular.nvim

Requires the latest neovim version (or neovim 0.10+ I think?)

Ubuntu install:
sudo add-apt-repository ppa:neovim-ppa/unstable -y
sudo apt update
sudo apt install make gcc ripgrep unzip git xclip neovim

# Old config
Rename init_old.vim to init.vim in order to use the old configuration. Then, name
init.lua to init_old.lua (or something else).
